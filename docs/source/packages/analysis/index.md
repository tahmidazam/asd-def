# analysis

The analysis package for the Litman stability workstream: reproducing the data-driven autism
classes of Litman et al. (2025) and testing whether they hold within strata of age at
diagnosis and diagnostic era.

The package is in early development, built one pipeline stage at a time. This page will
describe the stages and the workflow once they are implemented. For now, `analysis --help`
lists the planned stages, and the reference below documents the parts that exist: the
command-line entry point and the artefact-path helpers.
